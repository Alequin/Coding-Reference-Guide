Removing Commits

To remove the previous commit use the command:

git reset --hard HEAD^

To remove one or more commits use the command:

git reset --hard HEAD~n
 (n is the number of commits to remove)

(Sources:
sethrobertson.github.io: Removing the last commit )

Rename Branch

Rename the current branch.
git branch -m newname

Rename a different branch.
git branch -m oldname newname

(Sources:
StackOverflow: How to rename a local Git branch?)
