Base Constuctors

When inheriting from a super class the super classes constructor may be called from the sub classes using the keyword base.

public abstract class Mammal{

  private String voice;
  private String hair;

  public Mammal(String voice, String hair){
    this.voice = voice;
    this.hair = hair;
  }
}

public class Dog: Mammal{
  public Dog(String hair): base("woof", hair){}
}
