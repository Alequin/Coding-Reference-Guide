Output Parameters

Output parameters allow multiple values to be returned from a method using the parameters list rather than a return statement. If a method defines an out parameter if must be used like any other parameter.

public static int OutTest(int a, int b, out int c, out int d){
  c = a * a;
  d = b * b;
  return a + b;
}

int a = 0;
int b = 0;

int c = OutTest(2, 10, out a, out b);

Console.WriteLine(a);
Console.WriteLine(b);
Console.WriteLine(c);
//=> 4
100
12

As can be seen in the example using the out parameters does not mean a return statement cannot be used, though as always void may be used. All out parameters must be assigned though before the method ends.
