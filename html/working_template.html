Data Types

In C# there are three basic types.

  • Value types
  • Reference types
  • Pointer types

Value Types

Value types directly contain the data they are given and are derived from the class System.ValueType. Examples of value types are included at the bottom of this section.

Value Types Examples

  • bool
    • A true or false boolean value
    • default: false
  • char
    • A 16 bit unicode character
    • default: '\0'
  • int
    • A 32 bit interger
    • -2,147,483,648 to 2,147,483,647
    • default: 0
  • decimal
    • A 128 bit precise decimal value with 28-29 significant digits
    • (-7.9 x 1028 to 7.9 x 1028) / 10^(0 to 28)
    • default: 0.0M
  • long
    • A 64 bit integer
    • -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
    • default: 0L
  • short
    • A 16 bit integer
    • -32,768 to 32,767
    • default: 0
  • sbyte
    • A 8 bit integer
    • -128 to 127
    • default: 0
  • double
    • A 64 floating point number
    • (+/-)5.0 x 10-324 to (+/-)1.7 x 10308
    • default: 0.0D
  • float
    • A 32 floating point number
    • -3.4 x 1038 to + 3.4 x 1038
    • default: 0.0F
  • ushort
    • A 16 bit interger
    • 0 to 65,535
    • default: 0
  • uint
    • A 32 bit interger
    • 0 to 4,294,967,295
    • default: 0
  • ulong
    • A 64 bit interger
    • 0 to 18,446,744,073,709,551,615
    • default: 0
