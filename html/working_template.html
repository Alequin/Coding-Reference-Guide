Media Queries

Media queries are used in css to detect the types and size of the current screen and apply different styles.

@media only screen and (max-width: 480px) {
  body{
    font-size: 12px;
  }
}

This rule states that for all devices of type screen with a view port width of 480px or less will have a font size to 12px.

All rules with the media query will be applied as additional rules when the condition is met. All media query styles will override any originals regardless of priority.

Only Keyword

The keyword only is used to hide style sheets from old browsers which don't support media queries.

Device Type

The type of device can be one of four options...
Note: While there are others they are depreciated and should not be used.

  • all
  • print
  • screen
  • speech

Rules

There are a few rules that may be applied using a media query. A decent list can be viewed on w3shools section on the topic . Here are a few useful ones...

  • max-width
  • min-width
  • max-height
  • min-height
  • max-resolution
  • min-resolution
  • min-aspect-ratio
  • max-aspect-ratio
  • orientation

Note: Resolution rules use dpi or dpcm as units

Not Keyword

The keyword not may be used in queries to reverse the given rule. So a rule "max-width: 480px" used with the keyword not would be processed as "min-width: 480px".

The following two queries will produce the same results...

@media screen and (min-width: 500px){
  body{
    background: red;
  }
}

@media not screen and (max-width: 500px){
  body{
    background: red;
  }
}

Multiple Rules

Queries may use multiple rules to decide if styles should be applied. The 'and' keyword is used to require the device to meet two or more conditions before the styles are applied. Commas may be used to create queries which can be applied as long as one condition is met (similar to an OR statment).

@media only screen and (max-width: 320px) and (max-width: 480px) {
  body{
    font-size: 12px;
  }
}

@media only screen and (max-width: 480px), (orientation: portrait) {
  body{
    font-size: 12px;
  }
}
