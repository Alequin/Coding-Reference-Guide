Nullable

A Nullable is a value type that is able to hold a null value;

int? = null;
bool? = null;

There is also a Nullable operator which uses two operands and the operator '??'.

b = null;
c = b ?? 20;
Console.WriteLine(c);

b = 30;
c = b ?? 20;
Console.WriteLine(c);

//=> 20
30

As shown in the example if the first operand is null the second will be used (null ?? 20 returns 20). If the first operand is not null it will be returned (30 ?? 20 returns 30);
