Media Queries

Media queries are used in css to detect the types and size of the current screen and apply different styles.

@media only screen and (max-width: 480px) {
  body{
    font-size: 12px;
  }
}

This rule states that for all devices of type screen with a view port width of 480px or less set the font size to 12px.

All rules with the media query will be applied as additional rule when the condition is met.
