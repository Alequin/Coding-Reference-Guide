Dynamic testing

Dynamic testing is the opposite of static testing and involves testing code by running it, providing input and comparing the result against the expected value. Types of dynamic testing include unit testing, integration testing, system testing and acceptance testing.

Static Testing

Static testing (the opposite of dynamic testing) is a software testing method which involves testing the software without running the actual program. Static testing may conducted manually or with the aid of software and may include code analysis, inspection, code reviews and walkthroughs.

Errors found using this method may include undefined variables, unused code, unreachable code, security vulnerabilities, standards violations or syntax violations.
