Reference Types

Reference types do not contain actual data but the memory address to the required data. If a refence type is changed all varables that hold its memory location will refer to the changed value. Reference types in C# include object, dynamic and pointer.

  • Objects Types: This is the base type for all data types in C# Common Types System (CTS) and is an alias for the class 'System.Object'. The specific type of the object must be defined prior to compilation.
  • Dynamic Types: this type is able to hold any other type without declaring the what type is been stored. However when used it must be cast to it's required
  • Pointer Types: Pointer type variables store the memory address of another type.

Value Types

Value types directly contain the data they are given and are derived from the class 'System.ValueType'.

  • bool
    • A true or false boolean value
    • default: false
  • char
    • A 16 bit unicode character
    • default: '\0'
  • int
    • A 32 bit integer
    • -2,147,483,648 to 2,147,483,647
    • default: 0
  • decimal
    • A 128 bit precise decimal value with 28-29 significant digits
    • (-7.9 x 1028 to 7.9 x 1028) / 10^(0 to 28)
    • default: 0.0M
  • long
    • A 64 bit integer
    • -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
    • default: 0L
  • short
    • A 16 bit integer
    • -32,768 to 32,767
    • default: 0
  • sbyte
    • A 8 bit integer
    • -128 to 127
    • default: 0
  • double
    • A 64 floating point number
    • (+/-)5.0 x 10-324 to (+/-)1.7 x 10308
    • default: 0.0D
  • float
    • A 32 floating point number
    • -3.4 x 1038 to + 3.4 x 1038
    • default: 0.0F
  • ushort
    • A 16 bit integer
    • 0 to 65,535
    • default: 0
  • uint
    • A 32 bit integer
    • 0 to 4,294,967,295
    • default: 0
  • ulong
    • A 64 bit integer
    • 0 to 18,446,744,073,709,551,615
    • default: 0

Basic Structure

using System;
namespace HelloWorldApplication{

  class HelloWorld{

    static void Main(string[] args){
      Console.WriteLine("Hello World");
      Console.ReadKey();
    }
  }
}

  • using: the using keyword allows the code included in the provided namespace to be used in the current file.
  • namespace: this is where the namespace the current file is in should be defined.
  • class: this is the class declaration. A class contains the variables and methods that define the capabilities and behaviours of instances of that class.
  • static void Main(string[] args): this is the definition of the main method, which is the start point of all C# programs (the M in Main must be a capital).
