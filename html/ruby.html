Output to Console

print: print a string to the console
puts: print a string to the console with a new line after

Unique Operator

x <=> y: Returns 0 if x and y are equal, returns -1 if y is greater and 1 if x is greater

Loop with .times

5.times do puts "Hello" end
or
5.times {puts "Hello"}

The method .times runs the code in the provided block the specifed number of times. It can be called by integer values.
