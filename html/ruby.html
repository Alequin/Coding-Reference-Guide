Constants / Final Variables

To define a constant in Ruby create the variable with a capital as the first character

Dogs = 50

Any attemps to change the variable Dogs will result in a warning

Iterators

5.times do |number| puts "#{number}: Hello\n" end
5.times { |number| puts "#{number}: Hello" }

The method times runs the code in the provided block the specifed number of times. The example will iterate through numbers 0 - 4 and run the code in the given block

1.upto(5) do |number| puts "#{number}: Hello" end
1.upto(5) { |number| puts "#{number}: Hello" }
10.downto(5) do |number| puts "#{number}: Hello" end
10.downto(5) { |number| puts "#{number}: Hello" }

The methods upto and downto iterate through the given numbers inclusivly and run the code in the given block on each loop

0.step(50, 5) do |number| puts "#{number}: Hello\n" end
0.step(50, 5) { |number| puts "#{number}: Hello\n" }
3.step(50, 5) { |number| puts "#{number}: Hello\n" }

The method step will iterate through the given numbers inclusivly, in steps equals to the second parameter, and run the code in the given block on each loop. In the third example the final number will be 48 as the next value is greater than 50

(1..5).each do |number| puts "#{number}: Hello\n" end
(1..5).each { |number| puts "#{number}: Hello\n" }

Using a range the method each can be used to inclusivly iterate over the given numbers while running the code in the given block on each loop

Unique Operator

x <=> y: Returns 0 if x and y are equal, returns -1 if y is greater and 1 if x is greater

Output to Console

print: print a string to the console
puts: print a string to the console with a new line after
