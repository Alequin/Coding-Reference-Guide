Make Anchor Tags open a New Tab

Using the attribute target and providing a values of _blank will cause the link to open in a new tab.

< a href="www.website.com" target="_blank">website</a>

This however will create a vulnerable in your website as the linked site gains access to your page and is able to change the web address. This enables attacks to replace your site on the users browser with that delivers malware or one that look exactly the same in an attempt to gather personal information. This attack is known as phishing.

How phishing is achieved: the newly open site gains limited access to your page via the JavaScript window.opener object. This is a read/write object that they can manipulate. Among its properties is window.opener.location, which can be changed, causing the browser to go to a new URL.

Phishing may be avoided by including the attribute rel with a value of noopener but not all browsers support this and so some users will be at risk.

HTML Entities

In order to type reserved html characters entities are used. Entities normally have two identifiers (a name and a number) each of which can be used, though the number version is more likely to be supported by more browsers.
(Note: all "-" characters are used so that the numbers can be shown to you and are not part of the entities)

&-#60 / &-lt == <
&-#62 / &-gt == >
&-#38 / &-amp == &
