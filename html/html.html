HTML Entities

In order to type reserved html characters entities are used. Entities normally have two identifiers (a name and a number) each of which can be used, though the number version is more likely to be supported by more browsers.
(Note: all "-" characters are used so that the numbers can be shown to you and are not part of the entites)

&-#60 / &-lt == <
&-#62 / &-gt == >
&-#38 / &-amp == &
