What Is Unix

UNIX is an operating system first developed in 1969 and first published in 1971. UNIX systems provides a simple set of tools which perform simple and well defined functions within a unified file system. They also use shell scripting to combine tools to perform complex actions.

UNIX operating systems are made up of three parts...

The Kernal: This is the central component of the operating system and performs multiple functions. Some of the subsystems of the kernal include process management, scheduling, file management, device management, network management, memory management and device interupt managment.

The Shell: This is an interface between user and kernal which presents as a CLI (command line interpreter). The user inputs commands and the shell starts the associated program. Once the program has run the CLI prompt the user for more input.
