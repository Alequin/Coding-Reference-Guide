Merge Sort

The merge sort is a general purpose divide and conquer sorting algorithm that makes use of recursion. The sort works by dividing the given array of values in half multiple times untill you have a number of individual arrays, each with a length equal to your base case (usually 1).
Once the initial array has been divided the maximum number of times the values contained in the individual arrays are compared and merged to make increasingly larger sorted arrays. This is repeated untill there are no more arrays to merge. At this point the inital array should be sorted.
