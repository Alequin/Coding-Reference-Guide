Basic Patterns

  • Match any digit: \d
  • Match any non digit: \D
  • Match any character / wildcard: \.
  • Match any alphanumeric character: \w
  • Match any non-alphanumeric character: \W
  • Match any white space: \s
  • Match any non white space: \S
  • Match if one or more is present:
    • One or more digits: \d+
    • One or more white space: \s+
  • Match if zero or more is present:
    • Zero or more digits: \d*
    • Zero or more white space: \s*
  • Match if zero or one is present (optional character):
    • Zero or one digit: \d?
    • Zero or one white space: \s?
  • Match a range:
    • Match characters between a-z: [a-z]
    • Match characters between a-z and A-Z: [A-Za-z]
  • Match specific characters:
    • Match either an a,b or c: [abc]
    • Match either 3,5 or 7: [357]
  • Match everything but specific characters:
    • Match everything other than an a,b or c: [^abc]
    • Match everything other than 3,5 or 7: [^357]
  • Match text that starts with:
    • Match text that starts with an a,b or c: ^[abc]
    • Match text that starts with 3,5 or 7: ^[357]
  • Match text that ends with:
    • Match text that ends with an a,b or c: [abc]$
    • Match text that ends with 3,5 or 7: [357]$
