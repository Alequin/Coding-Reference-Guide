Exceptions and Error Handling

Exception allow code to manage uncontrollable situations and coding mistakes effectivly.

In Java the class Throwable is the superclass for all classes which are involved in error management. The three main subclasses of Throwable are Error, Exception and RuntimeException.

Error is used in situations in which a program should not recover from. They are abnormal conditions that are not commonly encountered.
Exception and its subclasses are used for anticipated issues which a program could recover from.
RuntimeException and its subclasses are used in general situations such as the array index being out of bounds.
