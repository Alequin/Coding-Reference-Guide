Interfaces

An interface is an abstract data type that defines a list static variables and abstract methods that the implementing classes must use. However, interfaces may be be left blank and used only as a data type. An interface is defined in the same way classes are only the word class is replaced with interface.

An interface may only define final static variables or public abstract methods. No methods in an interface may have a body and all methods must be overridden by the implementing classes, unless they are abstract. In the case of abstract classes the abstract methods definitions are inherited and don't have to be overriden until the first concreate class. They can be overridden it if required.

Once defined an interface is linked to a class with the keyword implements.

Note: the keyword abstract may be use in the initial definition of an interface but it is not required.
public abstract interface MyInterface{}

All method / variable definitons in the following example are vaild
// File - Swingable.java
public interface Swingable{
  public String MATERIAL_WOOD = "wood";
  public static final String MATERIAL_METAL = "metal";

  String swing();
  public String hit();
  public abstract String miss();
}

//File - Bat.java
public class Bat implements Swingable{

  public String swing(){
    return "Swinging the bat";
  }

  public String hit(){
    return "Made contact";
  }

  public String miss(){
    return "Its a miss";
  }
}

Interfaces may extend other interfaces. The child interface does not have to redefine the abstract methods but the must be used in the first concreate / non abstract class.

// File - Swingable.java
public interface Swingable{
  public String swing();
  public String hit();
  public String miss();
}

//File - Bat.java public interface Bat extends Swingable{}

//File - BaseballBat.java
public class BaseballBat implements Bat{

  public String swing(){
    return "Swinging the bat";
  }

  public String hit(){
    return "Made contact";
  }

  public String miss(){
    return "Its a miss";
  }
}
