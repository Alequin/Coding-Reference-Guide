Forms

HTML forms are used to collect information from a user. The input from the form may be used by front end javascript scripts or back end system. Forms offer elements such as text fields, checkboxes, radio buttons and drop down menus.

An important attribute for tags invloved with forms is name. Name is a method of identification for each section.

<form name="my_form"></form>
The tag form is the outer most tag of a HTML form.

<input type="text" name="text_input" />

<textarea name="text_area" rows="8" cols="80"></textarea>
text_area =>
The textarea is for larger amount of text. The rows attribute sets the max line height of the area. The user can change the height but it can never be less than the given number of lines. The cols attribute sets the width of the of the textarea.
