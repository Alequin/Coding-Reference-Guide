Forms

HTML forms are used to collect information from a user. The input from the form may be used by front end javascript scripts or back end system. Forms offer elements such as text fields, checkboxes, radio buttons and drop down menus.

An important attribute for tags invloved with forms is name. Name is a method of identification for each section.

<form name="my_form"></form>
The tag form is the outer most tag of a HTML form. It is not required but functionality may be lost without it.

<input type="text" name="text_input" />
<input type="password" name="password_text" />

The tag input can be used in a number of ways. Here the type attribute instructs the input element to accept text. If tpye is set to password all input will be hidden.

<textarea name="text_area" rows="8" cols="80"></textarea>
text_area =>
The textarea is for larger amount of text. The rows attribute sets the max line height of the area. The user can change the height but it can never be less than the given number of lines. The cols attribute sets the width of the of the textarea.

<input type="checkbox" name="check1" >check box 1</input>

The checkbox input is for when there are multiple options a user may select from.

<input type="radio" name="radio" >radio 1</input>

Radio buttons are similar to checkboxes. The only difference is that users may only select a single option. Radio buttons that are associated should be given the same value for the name attribute.

<select name="drop-list">
  <option value="option 1"></option>
  <option value="option 2" selected></option>
</select>

<select name="drop-list2" size="3">
  <option value="option 1"></option>
  <option value="option 2" selected></option>
</select>

The select and option tags are used to create a drop down list of selections. The attribute select (which has no value) is used to set the initially selected value. The attribute size decides the height of the list (a size of three reveals three options). If the size is less than the number of options a scroll bar will be provided.

<input type="submit" name="submit" value="submit button"/>
<input type="reset" name="reset" value="reset button"/>
<input type="button" name="button" value="button button"/>
<input type="image" name="image" src="../images/puppy.jpg"/>
Buttons in HTML forms can be created with the input tag and may be one of four types.
  • submit: The submit button will call the process defined by the action attribute, which belongs to the the form tag. It may also re-load the page.
  • reset: The reset button will set all elements within the same form as the button to their default state.
  • button: This has no pre defined function. Actions can be assigned with javascript.
